## ğŸ› Troubleshooting

### **Issue: Export button not appearing**

**Cause:** Response still streaming or no content

**Solution:**
- Wait for response to complete
- Check that response has content
- Refresh page if needed

---

### **Issue: Export fails with error**

**Cause:** Large response or browser memory

**Solution:**
- Try exporting smaller responses
- Close other tabs
- Use PDF instead of Word
- Clear browser cache

---

### **Issue: Formatting looks wrong**

**Cause:** Complex markdown or special characters

**Solution:**
- Simplify markdown in prompt
- Use basic formatting
- Export to Word and edit manually

---

### **Issue: File won't download**

**Cause:** Browser blocking downloads

**Solution:**
- Check browser download settings
- Allow downloads from localhost
- Disable popup blocker
- Try different browser
